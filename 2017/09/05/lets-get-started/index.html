Let's get started

2017/09/05

Several months ago, great Yihui Xie started a new package called blogdown for easy webpage and blogs creation with R markdown, which kept my attention. Since then many blogs that I follow as Hyndman, Romain François, Simply Statistics have switched to blogdown.

I have been blogging in Barcelona R Users group blog. At some point I stopped but I always missed it. It was a great way to share and learn things and to clarify ideas about what you write, doesn’t matter if it is about packages, functions or programming.

I never liked the WordPress way of doing things. As blogdown makes it very easy to create your blog, time has come to start my own.

So, what am I going to write about?

For sure it will be R related, Data Science and Machine Learning that right now I am fully interested in, but as it is my personal blog I could even write about history, cooking, trips, etc.

So let’s get started and we will see.
